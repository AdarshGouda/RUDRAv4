# RUDRA Setup Images

These are locally owned RUDRA article images used by
`../nuc_rudra_ros2_setup.html`.

The HTML keeps the Medium-exported filenames so the assets can be traced back to
the original article export. Images are grouped in the tutorial by progression:

- robot overview and v3 platform photos
- differential-drive theory diagrams
- control-loop and odometry flowcharts
- RUDRA architecture diagrams
- wiring, Sabertooth, PS2, encoder, and IMU references
- ROS/navigation stack diagrams

If images are renamed later, update the corresponding `assets/...` references in
`../nuc_rudra_ros2_setup.html`.
